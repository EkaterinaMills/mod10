Canvas Element Example

This web page uses the HTML5 <canvas> element to draw graphics directly on the page. In the example below, we draw a rectangle, a circle, and some text using the Canvas API.

About This Page

The canvas element is powerful for creating graphics, animations, games, and data visualizations. You can draw shapes, images, or text using JavaScript. In this example, we drew a few basic shapes to demonstrate its functionality.
